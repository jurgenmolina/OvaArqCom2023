Unidad: virtualizado, emuladores y Contenedores

En esta unidad se abordan los conceptos de virtualización, emuladores y contenedores y su relación con la arquitectura de computadores.

Recursos de investigación:

Recurso Descripción Tipo
VirtualBox Software de virtualización de código abierto para Windows, macOS, Linux y Solaris. Software
QEMU Emulador de procesador que permite ejecutar sistemas operativos completos. Software
Docker Plataforma de contenedores que permite empaquetar, distribuir y ejecutar aplicaciones en cualquier entorno. Software
VMware Workstation Software de virtualización para Windows y Linux. Software
Kubernetes Plataforma de orquestación de contenedores que automatiza la implementación, el escalado y la gestión de aplicaciones en contenedores. Software
Virtualización de hardware Tecnología que permite que múltiples sistemas operativos se ejecuten en un único hardware. Concepto
Contenedores Tecnología de virtualización que permite la ejecución de aplicaciones en entornos aislados y portables. Concepto
Emuladores de procesador Software que simula la arquitectura de un procesador en un entorno de software para permitir la ejecución de aplicaciones de manera aislada. Concepto

En resumen, esta unidad es fundamental para comprender cómo la virtualización, los emuladores y los contenedores están

Visualización, Emuladores y entornos de arquitectura

Este tema aborda la importancia de las herramientas de visualización y emulación en la arquitectura de computadoras, y cómo pueden ayudar en el diseño y desarrollo de sistemas. Algunos de los conceptos que se cubren son:

  • Emuladores de arquitectura de computadoras
  • Herramientas de visualización de sistemas
  • Entornos de simulación de sistemas
  • Comparación de diferentes herramientas y entornos

Este tema puede ser de gran ayuda para estudiantes de arquitectura de computadoras que deseen profundizar en la visualización y simulación de sistemas.

Carga de programas y Depuradores (Ensambladores y Desensambladores)

Este tema se enfoca en la carga de programas y la depuración de sistemas, utilizando herramientas como ensambladores y desensambladores. Algunos de los puntos que se abordan son:

  • Carga de programas en memoria
